# operator aritmatika

operator aritmatika diantaranya
  - penambahan (+)
  - pengurangan (-)
  - pembagian (/)
  - perkalian (*)
  - sisa bagi / modulo (%)
  - pangkat (**)


```php
10 // nilai positif
-10 // nilai negatif / min 10 , cukup tambahkan min untuk nilai negatif
$nilai1 = 10;
$nilai2 = 20;

$nilai1 + $nilai1 // 30
$nilai2 + $nilai1 // 10
$nilai1 * $nilai1 // 20
$nilai2 / $nilai1 // 20

$nilai2 ** 3 //800;

$nilai2 % 2 // 0;


```
